# Personal-Portfolio
A webpage designed as a personal portfolio/CV.

The webpage was designed using plain HTML & CSS for simplicity.

Page: https://mohamed-elbagir.free.je

### CSS
The first section under the "style" element defines variables that will hold the colors used in the webpage, so colors
can be simply changed by modifying these variables. The remaining code under the "style" element adjusts the font type, font size, colors, 
padding and other styling components of the HTML elements.

### HTML
The body of the page contains the header, navigation links, main and footer. The header displays the name and the role under it. Navigation links 
let the visitor jump directly to the corresponding section. Main contains the relevant information of the page, which is about, experience, education, skills, 
projects and contact links. Footer just displays the author name and copyright symbol.

The about section has a header and a paragraph under. Both the experience and eduation sections have headers and sub-headers under them 
to display a title and date, in addition the experience section has a paragraph under the header. The projects section is the same as the 
experience section but does not include a date. The skills section has a header and sub-headers each containing an unordered list of elements.
The contact section has three links, one to send an email, the other is for a phone number and the last is a Github page.
